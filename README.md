这个是一个基于 AI 的广告营销创意和方案生成平台

# 背景信息

## 行业方法论

基于整合营销传播全案的标准流程（遵循 5W2H 说服逻辑），广告营销创意方案的生成包含以下核心步骤：

### 标准十步法

1. **简报解读（Brief）** - 明确背景、目标、任务三要素，确定现状（Point A）到目标（Point B）的转变
2. **洞察分析（Insight）** - 五维度深度分析：环境洞察（PEST+SWOT）、行业洞察、企业洞察、竞品洞察、消费者洞察
3. **策略制定（Strategy）** - 核心是做出最优选择：对谁说、说什么、怎么说
4. **核心概念（Concept）** - 用一句话概括创意主题和核心传播主张（Big Idea）
5. **创意执行（Idea）** - 概念的趣味表达，涵盖多种形式（视频、图文、H5 等）和风格（幽默、情感、故事等）
6. **媒介规划（Media）** - 选择渠道组合：自有媒体、付费媒体、赢得媒体
7. **排期计划（Schedule）** - 制定时间轴和传播阶段：预热期 → 爆发期 → 持续期
8. **效果承诺（KPI）** - 设定营销目标和传播目标，包括曝光量、点击率、转化率、ROI 等指标
9. **预算分配（Budget）** - 详细拆分内容制作、渠道投放、KOL 合作等成本，预留 10-15%弹性空间
10. **执行保障（Company）** - 明确责任人、交付标准、合规审核

### 关键执行原则

- **目标一致性**：创意、渠道、内容紧扣核心目标
- **差异化与合规性**：避免同质化，严格符合广告法
- **数据驱动决策**：以数据为依据调整策略，避免凭感觉
- **媒介适配性测试**：大规模投放前进行小范围测试

# 平台功能

平台基于**大模型、AGENT、爬虫**等核心技术，为广告营销创意和方案生成提供强大的技术支撑。每个核心功能都深度融合了先进技术，相比传统工作模式，在**效率**和**质量**上带来巨大提升。

## 核心技术能力概览

- **大模型能力**：使用 BAAI/bge-large-zh-v1.5 向量模型进行语义理解与检索，支持内容生成、知识推理与决策
- **AGENT 能力**：智能工作流引擎、多模态文档处理、自动化分析引擎、人机协作机制
- **爬虫能力**：多平台数据采集、智能爬取策略、数据处理管道、采集监控与管理
- **技术架构**：基于 FastAPI 的异步架构、PostgreSQL + pgvector 向量数据库、分布式可扩展架构

- 自动解析 brief 文件中的背景信息、目标设定、任务要求
- 识别 Point A（现状）和 Point B（目标）
- 提取关键信息并结构化输出

- 支持 PDF、Word、Excel、PPT、图片、音频、视频等多种格式的智能解析
- 自动提取文本、表格、图表、图片等结构化信息
- 文档质量检测，识别清晰度、完整性、可读性

**大模型智能解读**：

- 基于大语言模型自动解析 Brief 文件，提取九大核心信息（背景、目标、任务等）
- 深层需求挖掘和痛点识别
- 自动识别 Point A（现状）和 Point B（目标）的转变路径

**AGENT 工作流引擎**：

- 基于标准化流程（Brief 解读六步法）自动编排工作流
- 每个节点的输入输出验证和状态跟踪
- 异常情况自动识别和预警

### 效率与质量提升

- **效率提升**：将 Brief 处理时间从数小时缩短至数分钟，自动化程度达 80% 以上
- **质量提升**：基于标准化流程确保分析结果的专业性和一致性，减少人为遗漏和错误
- **成本优化**：减少人力投入，降低项目前期成本 60% 以上

## 2. 资料收集

### 核心功能

- **案例检索和收集**：基于平台案例库，检索相似案例、成功案例、竞品案例
- **行业研报**：收集行业趋势、市场规模、竞争格局等研究报告
- **品牌调性分析**：收集品牌历史、定位、调性、视觉风格等信息
- **用户人群研究**：收集目标受众的人口统计、行为习惯、心理诉求等数据
- **竞品分析**：收集竞品策略、创意、投放渠道等信息

**输出**：结构化的资料库，支持多维度检索和引用

### 技术能力支撑

**爬虫多平台数据采集**：

- 自动采集广告行业网站（广告门、数英网、SocialBeta 等）
- 采集社交媒体平台（小红书、抖音、B 站等）的营销案例
- 采集品牌官方平台和行业报告平台数据
- 使用 Playwright/Selenium 浏览器自动化，支持 JavaScript 渲染页面
- 智能反爬虫处理：代理 IP 轮换、请求频率控制、真实浏览器行为模拟

**大模型语义检索**：

- 使用 BAAI/bge-large-zh-v1.5（1024 维）向量模型进行语义理解
- 基于向量相似度的语义搜索，理解用户意图而非仅关键词匹配
- 混合检索：结合关键词检索和语义检索，提供更精准的案例匹配
- 支持中英文混合场景的语义理解

**数据处理管道**：

- 自动验证采集数据的完整性和准确性
- 数据清洗和标准化，智能去重
- 自动下载案例相关图片，支持批量处理和断点续传
- 增量采集：只采集新增和更新的内容

### 效率与质量提升

- **效率提升**：自动化采集替代人工搜索，7×24 小时持续更新，资料收集效率提升 10 倍以上
- **质量提升**：基于语义检索找到更相关的案例，覆盖范围更广，数据更及时准确
- **成本优化**：减少人工收集时间，降低资料收集成本 70% 以上

## 3. 策略分析

### 核心功能

- **五维度洞察分析**：

  - 环境洞察：PEST 分析（政治、经济、社会、技术）+ SWOT 分析
  - 行业洞察：市场规模、竞争格局、细分市场分析
  - 企业洞察：品牌定位、营销 4P 分析（产品、价格、渠道、促销）
  - 竞品洞察：对标分析、差异化定位
  - 消费者洞察：人口统计、行为习惯、心理诉求三维度分析

- **策略制定**：
  - 确定"对谁说"：目标受众画像和细分
  - 确定"说什么"：核心传播信息（3-5 条关键信息）
  - 确定"怎么说"：传播调性和表达方式

**输出**：策略分析报告，包含洞察结论和策略建议

### 技术能力支撑

**大模型知识推理与决策**：

- 利用大模型进行五维度洞察分析，自动生成 PEST+SWOT 分析框架
- 深层挖掘客户需求和痛点，量化评估项目风险和可行性
- 智能对比分析竞品策略和差异化定位
- 基于数据驱动的策略制定和决策支持

**AGENT 自动化分析引擎**：

- 基于 NLP 技术自动提取和分析关键信息
- 自动生成多维度洞察结论和策略建议
- 基于分析结果动态生成专业报告（PDF、Word、PPT 等格式）

**大模型语义检索增强**：

- 基于语义检索快速找到相关行业报告、竞品案例、消费者研究数据
- 智能关联和整合多源数据，形成完整的洞察分析

### 效率与质量提升

- **效率提升**：自动化分析替代人工研究，策略分析时间从 1-2 周缩短至 1-2 天，效率提升 5-10 倍
- **质量提升**：基于大模型的知识推理能力，分析更深入全面，减少主观判断偏差
- **数据支撑**：自动引用第三方数据和案例，增强分析结论的说服力和可信度
