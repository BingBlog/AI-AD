# AI 广告营销创意平台 - 技术方案

## 平台概述

AI 广告营销创意平台是一个基于**大模型、AGENT、爬虫**等核心技术的智能营销方案生成系统。平台遵循整合营销传播全案的标准流程（5W2H 说服逻辑），为广告营销创意和方案生成提供全流程技术支撑。

## 核心技术能力

### 1. 大模型能力

- **向量模型**: BAAI/bge-large-zh-v1.5（1024 维）用于语义理解与检索
- **内容生成**: 支持创意内容、策略分析、方案文档的自动生成
- **知识推理**: 支持多维度洞察分析和决策支持
- **语义检索**: 基于向量相似度的智能检索，理解用户意图

### 2. AGENT 能力

- **智能工作流引擎**: 基于标准化流程自动编排工作流
- **多模态文档处理**: 支持 PDF、Word、Excel、PPT、图片、音频、视频等格式
- **自动化分析引擎**: 基于 NLP 技术自动提取和分析关键信息
- **报告生成引擎**: 动态生成专业报告（PPT、PDF、Word 等格式）
- **人机协作机制**: 支持人工审核和智能推荐的协作模式

### 3. 爬虫能力

- **多平台数据采集**: 支持广告行业网站、社交媒体平台、品牌官方平台等
- **智能爬取策略**: 代理 IP 轮换、请求频率控制、真实浏览器行为模拟
- **数据处理管道**: 数据验证、清洗、标准化、智能去重
- **采集监控与管理**: 任务管理、进度监控、断点续传

### 4. 技术架构

- **后端**: FastAPI 异步架构，支持高并发处理
- **数据库**: PostgreSQL + pgvector 向量数据库，支持语义检索
- **前端**: React 18 + TypeScript + Ant Design，现代化 UI/UX
- **部署**: 支持 Docker 容器化部署，分布式可扩展架构

## 快速开始

### 环境要求

- Python 3.9+
- Node.js 18+
- PostgreSQL 14+
- pnpm 或 npm

**推荐使用统一启动脚本**：

./start.sh

- ✅ 向量模型配置
