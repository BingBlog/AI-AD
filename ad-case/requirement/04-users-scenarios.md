## 用户角色与场景

### 用户角色

1. **广告创意人员（主要用户）**

   - **角色描述**：负责广告创意设计和方案策划的专业人员
   - **使用频率**：高频使用，每天多次检索案例
   - **核心需求**：快速找到相关创意案例，了解行业趋势
   - **技能水平**：熟悉广告行业，对案例质量要求高

2. **策划人员**

   - **角色描述**：负责活动策划和方案设计的专业人员
   - **使用频率**：中高频使用，项目期间频繁使用
   - **核心需求**：查找同类型策划案例，对比分析多个案例
   - **技能水平**：熟悉策划流程，需要详细的案例信息

3. **项目经理/客户经理**

   - **角色描述**：负责项目管理和客户沟通的管理人员
   - **使用频率**：中频使用，需要时使用
   - **核心需求**：向客户展示案例，了解竞品案例
   - **技能水平**：对系统易用性要求高，需要快速上手

4. **团队管理者**
   - **角色描述**：负责团队管理和知识管理的管理人员
   - **使用频率**：低频使用，定期查看
   - **核心需求**：了解案例库建设情况，分析使用数据
   - **技能水平**：需要数据分析和统计功能

### 典型使用场景

1. **场景一：快速查找创意参考**

   - **用户**：广告创意人员
   - **场景描述**：接到新项目，需要快速查找同行业、同类型的创意案例作为参考
   - **操作流程**：
     1. 输入关键词或选择行业、标签进行检索
     2. 浏览检索结果，筛选出相关案例
     3. 查看案例详情，了解创意亮点
     4. 收藏有价值的案例，用于后续参考
   - **成功标准**：5 分钟内找到 3-5 个高质量相关案例

2. **场景二：对比分析多个案例**

   - **用户**：策划人员
   - **场景描述**：设计活动方案时，需要对比分析多个成功案例，提炼策划要点
   - **操作流程**：
     1. 按活动类型、品牌等维度检索案例
     2. 筛选出多个相关案例
     3. 对比分析案例的策划思路和执行方式
     4. 导出案例信息，用于方案设计
   - **成功标准**：能够快速对比分析多个案例，提炼出策划要点

3. **场景三：向客户展示案例**

   - **用户**：客户经理
   - **场景描述**：向客户提案时，需要展示相关的成功案例，增强方案说服力
   - **操作流程**：
     1. 按客户行业、品牌类型检索案例
     2. 筛选出高质量、有代表性的案例
     3. 查看案例详情和效果数据
     4. 分享案例链接或导出案例信息
   - **成功标准**：快速找到符合客户需求的优质案例

4. **场景四：了解行业趋势**
   - **用户**：广告创意人员
   - **场景描述**：定期浏览最新案例，了解行业创意趋势和优秀案例
   - **操作流程**：
     1. 按时间排序，查看最新案例
     2. 浏览热门标签和行业分类
     3. 关注特定品牌或广告公司的最新案例
     4. 收藏感兴趣的案例
   - **成功标准**：能够持续发现行业最新趋势和优秀案例
